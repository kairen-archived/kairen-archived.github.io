Posted on 2016-10-23
Kubernetes
Docker, Kubernetes

Minikube 是提供簡單與快速部署本地 Kubernetes 環境的工具，透過執行虛擬機來執行單節點 Kubernetes 叢集，以便開發者使用 Kubernetes 與開發用。

本環境安裝資訊：

  • Minikube v0.22.3
  • Kubernetes v1.7.5

事前準備

安裝前需要確認叢集滿足以下幾點：

  • 安裝 xhyve driver, VirtualBox 或 VMware Fusion。
  • 安裝 kubectl 工具。
